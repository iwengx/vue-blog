# HTML Symbols

## 介绍

HTML 中自带了不少的图标，可以通过输入它们的 **Dec** 或 **Hex** 值显示出他们（在 Markdown 中也有类似的图标显示）

做聊天系统的时候可以将表情的代码存入数据库即可。
